# Executive summary

## Main findings

- Test group conversion: 11.75%; control group conversion: 10.09%.
- Relative uplift in purchase conversion: 16.39%; p-value: 0.0441.
- Test group ARPU: 16.55; control group ARPU: 13.77.
- Relative uplift in ARPU: 20.18%; p-value: 0.0386.
- Highest revenue segment: organic / web with revenue 10823.00.

## Recommendation

The test variant shows stronger monetization metrics. Before full rollout, it is worth checking retention and support metrics for the same user groups, then launching the variant gradually for the best-performing channels and platforms.
